**Here are two simple HTML pages** that implement what you asked for: - `register.html` — Registration form - `login.html` — Login page ### How it works - Registration saves the username + password to the browser's **localStorage** (as a JSON array of users). This is the safest and simplest way for a pure frontend demo (real apps would use a server + database). - After successful registration, it automatically redirects to the login page. - Login verifies the credentials against the stored users. ### Files created **1. `register.html`** ```html Register - MyWebsite

Create Account

Already have an account? Login here

``` **2. `login.html`** ```html Login - MyWebsite

Login

Don't have an account? Register here

``` ### How to test it 1. Save both files in the same folder. 2. Open `register.html` in your browser. 3. Register an account → it will redirect to login. 4. Login with the same credentials. **Note**: This is a **frontend-only demo**. Passwords are stored in plain text in `localStorage` (not secure for production).
